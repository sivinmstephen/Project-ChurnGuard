# Project ChurnGuard: AI-Driven Revenue Assurance

## Executive Summary
Project ChurnGuard is a strategic predictive analytics solution designed to mitigate customer attrition in a database of 11,260 accounts. By transitioning from reactive troubleshooting to a proactive AI-driven framework, we identified key churn drivers and developed a model capable of catching 87% of potential churners before they exit.

## Business Impact
* **Revenue Protection:** Stabilizing churn in the "Regular Plus" segment is projected to recover over **$11,000 in monthly revenue**.
* **Strategic Insight:** Identified a critical "30-day onboarding red zone" and "Tier 3 infrastructure gap."

## Tech Stack
* **Language:** Python
* **Libraries:** Pandas, Scikit-Learn, XGBoost, Matplotlib, Seaborn
* **Environment:** Google Colab
